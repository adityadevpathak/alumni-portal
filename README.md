# Alumni Portal Web Application

A web-based alumni portal designed to connect college students with alumni for guidance, updates, and career opportunities.

## Features
- User registration and login
- Secure password authentication
- Alumni profile management
- Create and view posts
- Search alumni by batch and company

## Tech Stack
- Python
- Flask
- SQLAlchemy
- SQLite
- HTML, CSS

## Purpose
This project helps college students connect with alumni to receive career guidance, referrals, and insights from seniors.

## How to Run Locally
```bash
git clone https://github.com/adityadevpathak/alumni-portal.git
cd alumni-portal
pip install -r requirements.txt
